# 📘 Теория OpenCV для решения задач (№3–5)

## 🛠 Общая база OpenCV

Изображение в OpenCV — это многомерный массив библиотеки NumPy:
```python
height, width, channels = img.shape
```

**Система координат:**
Координаты всегда считаются от **левого верхнего угла**:
*   `x` → вправо
*   `y` ↓ вниз

**Цветовая модель:**
Обрати внимание: цвета в OpenCV задаются в формате **BGR**, а не RGB!
```python
(255, 0, 0)     # Синий
(0, 255, 0)     # Зелёный
(0, 0, 255)     # Красный
(255, 255, 255) # Белый
```

**Основные функции:**
```python
cv2.imread()           # Загрузка изображения
cv2.imwrite()          # Сохранение изображения
cv2.imshow()           # Показ изображения в окне
cv2.waitKey(0)         # Ожидание нажатия клавиши (чтобы окно не закрылось)
cv2.destroyAllWindows()# Закрытие всех окон
```

## 🎄 Задание 3: Зимнее дерево, снег, солнце, текст, шар

### 1. Отражение изображения по вертикальной оси
Вертикальная ось — это зеркалирование изображения слева направо. Для этого используется функция:
```python
cv2.flip(img, flipCode)
```
**Параметры `flipCode`:**
*   `1` — отражение по вертикальной оси (то, что нужно тебе).
*   `0` — отражение по горизонтальной оси.
*   `-1` — отражение и по вертикали, и по горизонтали.

### 2. Рисование круга
Для солнца и новогоднего шара тебе понадобится эта функция:
```python
cv2.circle(image, center, radius, color, thickness)
```
**Разбор параметров:**
*   **center** `(x, y)` — координаты центра.
*   **radius** — радиус круга в пикселях.
*   **color** — цвет в формате BGR.
*   **thickness** — толщина линии (если указать `-1`, круг будет полностью залит цветом).

💡 *Совет:* Золотистый цвет в BGR можно подобрать как оттенок жёлтого/оранжевого, например, близко к `(0, 215, 255)`. Экспериментируй!

### 3. Случайный снег
Снег — это много маленьких белых точек. Для этого нужна генерация случайных координат в пределах размера картинки:
```python
x = np.random.randint(0, width)
y = np.random.randint(0, height)
```
Каждая снежинка — это маленький белый круг (через `cv2.circle`) радиусом 1–3 пикселя.

### 4. Текст на изображении
Добавление надписи делается через:
```python
cv2.putText(image, text, org, fontFace, fontScale, color, thickness)
```
**Разбор параметров:**
*   **text** — сама строка.
*   **org** — координаты левого нижнего угла начала текста `(x, y)`.
*   **fontFace** — тип шрифта (например, `cv2.FONT_HERSHEY_SIMPLEX`).
*   **fontScale** — множитель размера.
*   **color** и **thickness** — цвет (BGR) и толщина линий.

💡 *Важно:* Чтобы надпись была точно по центру или занимала всю ширину внизу, нужно заранее вычислить её габариты. Для этого используй:
```python
cv2.getTextSize()
```
Она вернет ширину и высоту текста, что позволит правильно рассчитать координаты `org`.

### 5. Новогодний шар на ветке
Это просто комбинация двух функций:
1.  Рисуешь залитый круг `cv2.circle()`.
2.  Поверх него пишешь текст `cv2.putText()` (например, "NEW YEAR").
*Координаты шара придется подобрать вручную, ориентируясь на картинку дерева.*

## 🎞 Задача 4: Эффект старой фотографии
Здесь главная работа идет с цветом, масками, шумом и ручной отрисовкой дефектов.

### 1. Цветовое пространство HSV
Обычное изображение BGR неудобно для изменения насыщенности цвета. Поэтому его переводят в HSV:
```python
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)
```
**Что такое HSV:**
*   **H (Hue)** — оттенок цвета.
*   **S (Saturation)** — насыщенность (уменьши её, чтобы сделать фото блеклым).
*   **V (Value)** — яркость (немного уменьши, чтобы фото стало темнее/старее).

После изменения нужных каналов, не забудь вернуть картинку обратно:
```python
img_bgr = cv2.cvtColor(hsv, cv2.COLOR_HSV2BGR)
```

### 2. Виньетка
Виньетка — это затемнение по краям фото (центр светлый, края тёмные).
*   **Идея:** Создается маска размером с фото. В центре маски значения ближе к `1`, по краям — ближе к `0`.
*   **Инструменты:** Можно использовать `np.meshgrid()` для математического создания градиента или гауссовы ядра через `cv2.getGaussianKernel()`.
*   Затем исходное изображение перемножается с этой маской.

### 3. Зерно / шум плёнки
Шум — это случайные значения, добавленные к пикселям картинки.
*   **Инструменты:** `np.random.normal()` или `np.random.randn()`.
*   **Теория:**
    *   Маленькая дисперсия → лёгкий, естественный шум.
    *   Большая дисперсия → сильный шум.
*   *Осторожно:* Не переборщи, иначе фото превратится в "грязь".

### 4. Царапины
Царапины — это тонкие, хаотичные линии.
*   **Функция:** `cv2.line()`
*   **Признаки старых царапин:** они светлые, очень тонкие (толщина 1), расположены случайно, чаще всего идут вертикально или по диагонали.

### 5. Пятна
Для имитации пятен времени подойдут:
*   Эллипсы: `cv2.ellipse()`
*   Или круги: `cv2.circle()`
*   *Цвет:* Делай их не чисто белыми, а грязно-желтыми или серыми.

### 6. Осветление центра
Процесс, обратный виньетке: центр становится светлее, края не меняются.
Можно создать отдельную «маску яркости» и аккуратно наложить её поверх фото через:
```python
cv2.addWeighted()
```

## 🖼 Задача 5: Восстановление картины
В этой задаче на первый план выходят фильтры и математическая морфология.

### 1. Усредняющий фильтр (Blur)
Нужен для базового сглаживания мелкого шума:
```python
cv2.blur(img, ksize)
```
**Смысл:** Каждый пиксель заменяется средним значением всех соседних пикселей в окне `ksize`.
**Размеры окна `ksize`:**
*   `(3, 3)` — слабое размытие.
*   `(5, 5)` — среднее (рекомендуется в задании).
*   `(7, 7)` — сильное (пропадают важные детали).

### 2. Эрозия (Erosion)
Уменьшает светлые области. Отлично подходит для «съедания» мелкого белого шума или дефектов на темном фоне.
```python
cv2.erode(img, kernel)
```
**Kernel** — это матрица (ядро), определяющая область воздействия. Пример создания:
```python
kernel = np.ones((3, 3), np.uint8)
```

### 3. Дилатация (Dilation)
Противоположность эрозии. Расширяет светлые области.
```python
cv2.dilate(img, kernel)
```
Если ты сначала применил эрозию (чтобы убить шум), объекты на фото тоже "похудеют". Дилатация вернет им исходную форму.

### 4. Морфологические операции (Opening и Closing)
Это готовые комбинации эрозии и дилатации, встроенные в OpenCV через `cv2.morphologyEx()`.

*   🟢 **Opening (Открытие):** `Эрозия → Дилатация`
    *   *Зачем:* Идеально для удаления мелкого точечного шума снаружи объектов.
*   🔴 **Closing (Закрытие):** `Дилатация → Эрозия`
    *   *Зачем:* Идеально для «замазывания» мелких дырок и трещин внутри объектов.

### 5. Главный баланс восстановления
Реставрация — это всегда компромисс:
*   **Слабая обработка:** на картине останется мусор и царапины.
*   **Сильная обработка:** картина превратится в "мыло", исчезнут текстуры и мазки художника.

Твоя задача — подобрать идеальный размер ядра (обычно `(3, 3)` или `(5, 5)`) и визуально оценить результат.
