# 🎨 Задача 9 — карандашный набросок через Sobel

**Главная идея:**
> фото → серое изображение → границы Sobel → объединение границ → инверсия → рисунок

### Что нужно знать

#### 1. Grayscale
Фильтры границ работают с яркостью, поэтому цветное изображение переводится в оттенки серого.
*   **цветное изображение:** BGR, 3 канала
*   **серое изображение:** 1 канал яркости

#### 2. Градиент
Граница объекта — это место, где яркость резко меняется.
*   тёмное → светлое
*   светлое → тёмное

Фильтр Собеля ищет такие перепады.

#### 3. Sobel по X и Y
Есть два направления:
*   **Sobel X** — ищет вертикальные границы
*   **Sobel Y** — ищет горизонтальные границы

**Параметры:**
*   `dx=1, dy=0` → производная по X
*   `dx=0, dy=1` → производная по Y
*   `ksize` → размер ядра фильтра

> Чем больше `ksize`, тем сильнее сглаживание, но тем больше теряются мелкие детали.

#### 4. Объединение границ
Чтобы получить общий контурный рисунок, нужно объединить результат по X и Y.  
**Идея:** общие границы = часть Sobel X + часть Sobel Y.  
Для этого обычно используют взвешенное сложение.

#### 5. Приведение типа
Sobel часто даёт результат в `float64`, а изображение для показа должно быть ближе к `uint8`.  
**Важно понимать:**
*   `float64` — удобно считать
*   `uint8` — удобно показывать и сохранять

#### 6. Инверсия
Для карандашного эффекта нужно:
*   **обычные границы:** белые линии на чёрном фоне
*   **карандашный рисунок:** чёрные линии на белом фоне

То есть цвета надо инвертировать.

# 🚗 Задача 10 — умный парктроник через Canny

**Главная схема:**
> изображение дороги → grayscale → GaussianBlur → Canny → контуры → фильтрация → наложение на оригинал

### Что нужно знать

#### 1. Почему нужно размытие
Canny чувствителен к шуму. Асфальт, мелкие камни, текстуры и блики могут дать лишние границы.  
Поэтому перед Canny делают **GaussianBlur**. Он сглаживает мелкий шум, но сохраняет крупные объекты.

#### 2. Canny
Canny использует два порога:
*   `threshold1` — нижний порог
*   `threshold2` — верхний порог

**Логика:**
1.  выше верхнего порога → сильная граница
2.  ниже нижнего порога → шум
3.  между порогами → граница только если связана с сильной

> Малые пороги дают много границ, большие — меньше границ.

#### 3. Наложение границ на оригинал
Canny выдаёт чёрно-белую маску границ. Чтобы показать границы на исходном фото, нужно:
*   найти пиксели границ или найти контуры
*   и нарисовать их поверх копии исходного изображения

**Используются идеи:**
*   `copy` исходного изображения
*   `drawContours`
*   цветная линия поверх фото

#### 4. Контуры
После Canny можно найти контуры:
*   **контур** = цепочка точек, образующих границу объекта.
Дальше их можно фильтровать.

#### 5. Фильтрация по площади
Мелкие контуры обычно являются шумом:
*   **маленькая площадь** → игнорировать
*   **большая площадь** → возможно препятствие

Площадь контура помогает отсечь мусор.

#### 6. Прямоугольные препятствия
Корпуса машин, знаки, стены часто имеют прямоугольные формы.  
Для этого используют упрощение контура:
> сложный контур → многоугольник с меньшим количеством вершин

Если после упрощения получилось примерно 4 вершины, это может быть прямоугольник.

#### 7. Ограничивающая рамка
Даже если контур не идеально прямоугольный, можно построить рамку вокруг объекта:
*   `x, y` — левый верхний угол
*   `w, h` — ширина и высота

Это удобно для визуализации препятствий.
